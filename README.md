## Installation

### RubyGems

```
$ gem install fluent-plugin-bind
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-bind"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format parser bind
```
